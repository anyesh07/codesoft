# CODSOFT — Python Internship Tasks

Completed by: **Anyesh Ojha**

This repository contains my solutions for the CodSoft Python Programming Internship.  
Each task is implemented as a fully independent Python script that can be run directly from the command line.

---

## Tasks Overview

### Task 1 — To-Do List
A simple command-line to-do list manager that allows adding, viewing, updating, and deleting tasks.  
**File:** `todo.py`

### Task 2 — Calculator
A basic calculator supporting addition, subtraction, multiplication, and division.  
**File:** `calculator.py`

### Task 3 — Password Generator
Generates a strong random password of user-defined length and complexity.  
**File:** `password_generator.py`

### Task 4 — Rock-Paper-Scissors Game
A console game where the user plays rock-paper-scissors against the computer.  
**File:** `rps.py`

### Task 5 — Contact Book
A contact management system that stores, searches, updates, and deletes contact details.  
**File:** `contact_book.py`
